Présentation

Je suis étudiante en BTS Services Informatiques aux Organisations, option SLAM (Solutions Logicielles et Applications Métiers). Ce portfolio présente mon parcours, mes compétences et les réalisations effectuées durant ma formation.

Compétences BTS SIO

Support et mise à disposition de services informatiques

Cette compétence consiste à installer, configurer, maintenir et mettre à disposition des services informatiques pour les utilisateurs. Elle comprend notamment la gestion des incidents, les sauvegardes, les droits d’accès, les systèmes Windows et Linux, ainsi que la documentation des interventions.

Développement de solutions applicatives

Cette compétence consiste à concevoir, développer et maintenir des applications. Elle mobilise les bases de la programmation, la programmation orientée objet, le développement web, les bases de données, SQL, PHP et les bonnes pratiques de développement.

Travail en mode projet

Le travail en mode projet permet d’organiser les tâches, de respecter les délais, de suivre l’avancement et de collaborer avec d’autres personnes autour d’un objectif commun.

Projets et réalisations

Création d’un site web HTML / CSS

Réalisation d’un site web simple en HTML et CSS afin de comprendre la structure d’une page web, les balises HTML et la mise en forme avec une feuille de style.

Compétences utilisées : HTML, CSS, organisation du code, publication web.

Gestion d’incidents avec GLPI

Utilisation de GLPI pour créer, suivre et traiter des tickets d’incidents. Cette réalisation permet de comprendre le cycle de vie d’un incident : déclaration, qualification, traitement, résolution et clôture.

Compétences utilisées : support utilisateur, ITIL, gestion de parc, documentation.

Base de données et requêtes SQL

Création de tables, ajout de données et écriture de requêtes SQL pour interroger une base de données. Cette réalisation permet de comprendre l’organisation des données et les relations entre les tables.

Compétences utilisées : modèle relationnel, SQL, clés primaires, clés étrangères.

Veille technologique

Ma veille technologique porte sur un thème lié à l’informatique, par exemple la cybersécurité, le cloud computing ou l’intelligence artificielle. Elle consiste à rechercher régulièrement des informations fiables, à les analyser et à les conserver.
